SimonSays Game

on a windows logo

The Simon Says Game is created using pure HTML, CSS, and Javascript.

Each turn increases the amount of tiles that light up and the player must memorize which tiles were pressed and the order in which they were pressed. The game tracks the amount of successful turns and ends when the player reaches 85 consecutive turns (which would be a new world record)
