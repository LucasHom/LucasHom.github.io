Pixabay Image Finder

Search for Ladybugs

The Pixabay Image Finder utilizes Pixabay API and Tailwind CSS to create a fully functional search bar and corresponding results.

The search function allows for any image filtered to be SFW and highly popular on the Pixabay Image list to ensure the images tailor well to the search request.

The search results list the number of views, downloads, likes, image type, and Pixabay ID. The owner of each image is provided through "Photo by" which also serves as a link to their Pixabay profile.
